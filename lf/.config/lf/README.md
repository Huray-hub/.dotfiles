# lf config

My lf config with thumbnail support

## Installation
```bash
chmod +x install.sh && ./install.sh
```

If you do not use my lfrc file add to yours:
```
set previewer ~/.config/lf/preview
set cleaner ~/.config/lf/cleaner
```
